## Что это за репо (15–60 сек)

[ГОЛОС] Meetily — ассистент для рабочих встреч. Пишет звонок, расшифровывает речь в текст, делает саммари. Идея понятная и, честно говоря, хорошая: все эти Otter, Fireflies и прочие «AI-заметки со встреч» гоняют твои переговоры через чужие серверы. А если ты юрист, врач или подписал NDA — это не «удобно», это проблема.

[ГОЛОС] Meetily обещает делать то же самое, но целиком на твоей машине. Rust, Tauri, Whisper и Parakeet локально, база — SQLite у тебя на диске. README прямым текстом: «No data ever leaves your computer».

[ГОЛОС] Проекту девятнадцать месяцев, пятьсот пятьдесят шесть коммитов. Это не поделка на выходные. Но вот что интересно: последний коммит — пятого июня. Больше месяца назад. А звёзды прилетели на этой неделе.

[АНИМАЦИЯ A1: StarChart — рост звёзд]

## Где README врёт

[ГОЛОС] А теперь разберём витрину. Три обещания в описании. Пойдём по одному.

[ГОЛОС] «Сто процентов локально, ни байта не уходит с вашего компьютера». Открываем бинарь.

[СКРИНКАСТ #7: strings по бинарю с грепом — на экране адрес PostHog; сразу рядом строка README «No data ever leaves your computer»]

[ГОЛОС] PostHog. Продуктовая аналитика, зашитая в приложение, которое продаёт себя обещанием, что данные не уходят никуда. Сейчас она выключена по умолчанию — я это подтвердил, соединений нет. Но давайте спросим гит, когда она стала выключена.

[СКРИНКАСТ #8: git log с поиском по строке миграции аналитики — на экране коммит от шестого мая: «fix: make analytics opt-in by default»]

[ГОЛОС] Шестое мая. С февраля по май телеметрия была включена **по умолчанию**. Всё это время README обещал ровно то же самое, что и сегодня. Они это починили сами, никто их за руку не ловил — и это в плюс. Но формулировку в README с тех пор не поправили ни на слово. Она была ложью три месяца и осталась в том же виде.

[ГОЛОС] Плюс мелочь, которая не мелочь: суммаризация умеет ходить в Claude, Groq и OpenAI — они зашиты рядом. Локально — только если сам выберешь Ollama. «Ни байта в облако» — правда лишь про одну конкретную конфигурацию, а написано как про продукт.

[ГОЛОС] Обещание номер два. «Распознавание спикеров». Кто говорит и когда — киллер-фича для расшифровки встреч.

[СКРИНКАСТ #9: рядом в одном кадре — строка описания репы со словами «speaker diarization» и строка README «Speaker diarization is planned for PRO in mid-June»; поверх — календарь с текущей датой]

[ГОЛОС] Её нет. Совсем. README сам, дважды, признаётся: фича запланирована. В платный тариф. На середину июня. На дворе июль, а последний коммит в репозитории — пятого июня.

[ГОЛОС] То есть в витрине висит как готовое то, чего не существует, что вдобавок будет платным, и что просрочено на месяц. Это не «README врёт» — это врёт строка, которую видят все.
